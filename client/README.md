This is a minimal client stub description.

You can generate a TypeScript/Python/Java client from docs/openapi.yaml using tools like OpenAPI Generator or Swagger Codegen.

Example (OpenAPI Generator CLI):

1. Install: `npm i -g @openapitools/openapi-generator-cli`
2. Generate: `openapi-generator-cli generate -i docs/openapi.yaml -g typescript-fetch -o client/typescript`
